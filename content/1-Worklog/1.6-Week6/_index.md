### Week 6: CLOUD COMPUTING AND BLOCK STORAGE (EC2 & EBS)

**Objective:** Deploy and operate virtual servers with a durable data backup mechanism.

**Detailed Content & Time:**
- **Monday (1.5h):** Khởi tạo máy chủ EC2 (Linux); tìm hiểu cách chọn Instance Type phù hợp với hiệu năng yêu cầu.
- **Tuesday (1.5h):** Quản lý EBS Volume: Thực hành gắn thêm ổ đĩa và thay đổi kích thước dung lượng.
- **Wednesday (2h):** Triển khai cơ chế sao lưu bằng EBS Snapshots và tìm hiểu cách phục hồi dữ liệu từ bản sao.
- **Thursday (1.5h):** Sử dụng User Data để tự động hóa việc cài đặt Web Server (Apache/Nginx) khi khởi động máy.
- **Friday (1h):** Thực hành quản lý trạng thái máy chủ (Start/Stop/Terminate) và theo dõi chi phí liên quan.

**Completion Date:** 13/02/2026

**Result:** Successfully operated virtual servers; mastered storage resource management skills.
